# CLAUDE.md

Claude Code のグローバル設定ファイルです。すべてのプロジェクトに共通して適用されます。

---

## コミュニケーション

- **チャットは日本語**でやり取りする
- **技術用語は英語**のまま使って構わない（例: `commit`, `branch`, `refactor`, `dependency` など）

---

## コーディング原則

### シンプルさを優先する
- 過度な設計・抽象化を避ける
- 現在の要件を満たす最小限の実装にとどめる
- 将来の仮想的な要件のために設計しない

### 変更範囲を最小化する
- 依頼された変更のみを行い、関連しない改善は加えない
- バグ修正の際に周辺コードをリファクタリングしない
- docstring・コメント・型アノテーションは変更したコードにのみ追加する

### セキュリティ
- SQL injection, XSS, command injection などの OWASP Top 10 脆弱性を導入しない
- secret や credential をコードにハードコードしない

---

## ファイル操作

- README やドキュメントファイルは明示的に依頼された場合のみ作成する
- 不要になったコードは削除する（コメントアウトや後方互換 shim を残さない）

---

## Git

- `.env` や credential ファイルは絶対に commit しない
- force push は明示的な指示がある場合のみ実行する
